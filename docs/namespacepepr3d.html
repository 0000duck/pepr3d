Detailed Description

Fix for CGAL bugs in polygon with holes verification Contained in CGAL PR #3688, likely to be in 4.14 version of CGAL.

Modified for Pepr3D to prevent flipping of camera.

For now, just keep this custom patched include file first, to avoid using CGAL's version
